# trivia-earn-league

Product Description: Play to earn online trivia competitions leveraging blockchain technology to provide a play-to-earn platform where users pay an entry fee to participate in Sports-centric-themed trivia contests. Players earn points with correct answers, and top scorers win cash prizes funded by the entry fees.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/trivia-earn-league.git
cd trivia-earn-league
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
